# Angular Basic Demo App

A simple Angular application demonstrating core features including routing, components, and two-way data binding.

## Features

- **Routing**: Navigation between multiple pages (Home, About Us, Services, Contact Us, Profile)
- **Two-Way Data Binding**: Live form preview in the Profile page using `[(ngModel)]`
- **Reusable Components**: Person component and Person list component
- **Responsive Design**: Clean UI with styled components

## Pages

- **Home**: Landing page
- **About Us**: Information about the application
- **Services**: Services overview
- **Contact Us**: Contact information
- **Profile**: Form with live preview of user input
